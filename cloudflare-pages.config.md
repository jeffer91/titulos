# Configuración rápida Cloudflare Pages

Usa estos valores al crear el proyecto en Cloudflare Pages:

```text
Project name: titulos
Production branch: main
Framework preset: None
Build command: vacío
Build output directory: /
Root directory: /
```

Después de publicar, abre primero:

```text
/
```

Luego prueba:

```text
/estudiantes
/coordinadores
/administradores
/seguridad
```
