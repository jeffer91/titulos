Pasos para publicar Apps Script

  1. Pega el código completo en Apps Script.
  2. Ejecuta una vez pruebaCrearHojas.
  3. Ve a Implementar → Nueva implementación.
  4. Tipo: Aplicación web.
  5. Ejecutar como: Yo.
  6. Acceso: Cualquier usuario con el enlace.
  7. Copia la URL terminada en /exec y pégala aquí.
